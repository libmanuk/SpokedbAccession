# SpokedbAccession
Creates valid SPOKEdb accession numbers for interviews and auto-fills the Interview Accession field.

## Accession Format

YYYY+oh+4digitaccessionforyear+_+projectcode+4digitaccessionforproject
# Example: 2019oh0001_dab0001
